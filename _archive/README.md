# Archive Directory

This directory contains code that has been identified as dead code but preserved for reference.

**DO NOT** import from this directory in production code.

## Contents

### frontend/
Archived frontend components that are no longer used:
- `E01V3Test.tsx` - Standalone EWF/E01 test component
- `E01V3Test.css` - Styles for E01V3Test
- `RawTest.tsx` - Standalone RAW disk image test component
- `ComponentPlayground.tsx` - Development component showcase
- `EvidenceTreeLazy.tsx` - Superseded lazy-loading tree
- `EvidenceTreeRouter.tsx` - Unused tree router

## Restoration

If you need to restore any of this code:

1. Move the file back to its original location
2. Update imports as needed
3. Remove from this archive
4. Update `docs/DEAD_CODE.md`
